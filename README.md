## 🚀 快速开始 (使用方法)

在 Termux 中复制并运行以下命令即可一键安装：

```bash
bash <(curl -sL https://raw.githubusercontent.com/weiranxinyu/ST-beilu-Rapid_deployment/main/ST-Manager/install.sh)
```

安装完成后，输入以下命令启动管理菜单：

```bash
st-menu
```
